NutriFit Tracker - Starter Prototype (IT488)

This starter project can be opened in IntelliJ IDEA as a simple static web project.

Files:
- index.html       : Main UI for logging meals and workouts and viewing a summary dashboard.
- tests.html       : Simple test runner that exercises core JavaScript functions and prints results to the console.
- css/styles.css   : Basic responsive styling.
- js/app.js        : Application logic (state, addMeal, addWorkout, calculateTotals, DOM rendering).
- js/tests.js      : Basic test statements that demonstrate verification of business logic.

How to run:
1. Open the project folder in IntelliJ IDEA.
2. Right-click index.html and select 'Open in Browser' to view the app.
3. Right-click tests.html and open in a browser to execute tests; view results via the browser console.
