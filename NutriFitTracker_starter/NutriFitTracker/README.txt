NutriFit Tracker - Enhanced Prototype (IT488)

This project can be opened in IntelliJ IDEA as a simple static web project.

Files:
- index.html       : Main UI for logging meals and workouts and viewing a summary dashboard.
- tests.html       : Comprehensive test runner with visual interface and console output.
- run-tests.js     : Node.js command-line test runner for automated testing.
- package.json     : NPM configuration with test scripts.
- css/styles.css   : Basic responsive styling.
- js/app.js        : Application logic (state, addMeal, addWorkout, calculateTotals, DOM rendering).
- js/tests.js      : Comprehensive test suite validating calculations and core behaviors.

How to run:
1. Open the project folder in IntelliJ IDEA.
2. Right-click index.html and select 'Open in Browser' to view the app.
3. Right-click tests.html and open in a browser to execute tests with visual output.
4. For command-line testing: run 'npm test' or 'node run-tests.js'

Test Coverage:
- Meal operations (add, update, delete)
- Workout operations (add, update, delete)
- Calorie calculations and net totals
- Edge cases and error handling
- Input validation and type conversion
